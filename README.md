# Explorando-IA-Generativa-em-um-Pipeline-de-ETL-com-Python

## Sobre o Projeto:

Este projeto foi desenvolvido como parte do desafio da DIO com o objetivo de aplicar o processo de ETL (Extract, Transform, Load) utilizando Python e Inteligência Artificial.

O foco do desafio foi compreender como os dados percorrem cada etapa do pipeline, desde a obtenção dos dados até a geração de informações enriquecidas por IA.

---

## Objetivo:

* Extrair dados de usuários;
* Transformar os dados utilizando um modelo de IA generativa;
* Carregar os resultados enriquecidos na estrutura de dados utilizada pelo projeto.

---

## O que foi usado:

* Python
* Google Colab
* OpenAI SDK
* Groq API

---

## Conhecimentos Aplicados:

* ETL (Extract, Transform, Load)
* Consumo de APIs
* Engenharia de Prompt
* Manipulação de Dados com Python
* Inteligência Artificial Generativa
